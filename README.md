# Adobe Demo Commerce

Adobe Analytics / Customer Journey Analytics のデモ用に作成した静的サイトです。

## Pages

- `index.html`
- `product-a.html`
- `product-b.html`
- `product-c.html`
- `order1.html`
- `order2.html`

## Features

- 既存サイトと同じ 6 ページ構成
- `adobedtm` の Launch スニペットを既存値のまま維持
- `window.adobeDataLayer` を使ったページビュー、CTA、リード送信、購入イベント
- GitHub Pages でそのまま公開可能な静的ファイル構成

## Publish on GitHub Pages

1. `main` ブランチへ push
2. GitHub の repository settings で Pages を有効化
3. Source を `Deploy from a branch` / `main` / `/ (root)` に設定

公開URLの例:

`https://akutagawamasaaki.github.io/demosite/`
